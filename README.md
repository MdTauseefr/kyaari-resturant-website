# KyaaArt Login Page

A modern and stylish login page created using **HTML**  **CSS** and **js**.
This project features a custom **Kyaari website developement**, clean UI, responsive layout, and smooth styling for a creative website login interface.

## ✨ Features

* Custom KyaaArt logo
* Modern login page UI
* Responsive design
* Clean and simple layout
* HTML CSS & jS Only

## 🛠️ Technologies Used

* HTML5
* CSS3
* Javacripts

## 📂 Project Structure

```plaintext
project-folder
│
├── index.html
├── style.css
└── kyaaart-logo.png
```

## 🚀 How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
